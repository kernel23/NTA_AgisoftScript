  • Move Directory: The archive folder where processed image subfolders will be relocated once finished.

üöÄ Auto-Processing & Completion

After selecting the final folder, the script takes over! For each folder, it will automatically:

  • Align cameras, build a DEM, and generate an Orthomosaic.
  • Export the .tif and .kmz files to your Output folder.
  • Move the original images to your Move folder.
  • Clean up temporary project files to save space.

When everything is finished, a final pop-up box will appear telling you exactly how many folders were successfully processed!
